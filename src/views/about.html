About

GG collects and analyzes Minecraft PvP minigame match data from Overcast Network.

Data Collection

GG does not analyze every match - it only collects enough matches to produce meaningful statistics about maps and servers. (In other words, the minimum match length for a given match is almost definitely not the official "rush" record.)

Each day, a bunch of matches are added and the stats are all updated. Matches will probably be collected in a rolling fashion (e.g. after a couple weeks, the oldest matches in the database will be overwritten by new ones), but I need to play around with that a bit first.

There are many more cool features to come, stay tuned!

If you have a problem or suggestion, please feel free to open an issue on GitHub.

Technical Bits

This runs on Google's App Engine and is written in Python and jinja2, with Bootstrap css/js. Beautiful Soup is used to collect match information. Source.
