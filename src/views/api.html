API

Map and server pages can return json representations by adding the query json=true to the url. Try it out.

Note:
This API is not meant for heavy use. Due to the light nature of this app's hosting ("free"), you could very easily bring it to a hault with too many requests. If you plan on using this API for anything other than playing around (e.g. regularly grabbing lots of data with a script), please contact me first.
